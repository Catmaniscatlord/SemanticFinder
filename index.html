SemanticFinder

Frontend-only live semantic search with transformers.js

Semantic search right in your browser! Calculates the embeddings and cosine similarity client-side without server-side inferencing, using a quantized version of sentence-transformers/all-MiniLM-L6-v2.
Demo for this tutorial, building two former ones [1], [2] about vector search. Download this page.
For a more refined search, enter a low character count; for a coarser search, enter a high one.
